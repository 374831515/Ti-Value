Command to generate genesis_json.cpp files

Command:
bts_genesis_to_bin.exe "--json=./genesis_bts.json" "--prologue=./binary_genesis.prologue" "--epilogue=./binary_genesis.epilogue" "--source-out=./genesis_json.cpp"



This version of bts_genesis_to_bin.exe will generate init_genesis.json file and corresponding keys.json and wif_keys.json files, and also it will generate genesis_json.cpp, put genesis_json.cpp under D:\GRBit_ver1\grbit\libraries\blockchain
